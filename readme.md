# **LLWP** Loomis-Wood Plots

LLWP is a fast and easy software to assign lines in spectra, which relies especially on Loomis-Wood plots. It is written in Python but also available as an *.exe* file, making installing on windows as easy as downloading a single file.

Visit its website [here](https://ltotheois.github.io/LLWP/index.html).
