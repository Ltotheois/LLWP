LLWP is a fast, efficient and easy solution for exploring and assigning spectra, which relies on Loomis-Wood plots. It is available as a Python script or an .exe file and thus can be used on all major operating systems.

If you are already convinced head to the Download section. Also, see the Docs and Video Explanations for the documentation and useful information. For other concerns contact me.
