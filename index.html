LLWP

LLWP is a fast and easy software to assign lines in spectra, which relies especially on Loomis-Wood plots. The following video demonstrates the assignment of 30 lines in about 30 seconds.

LLWP is a Loomis-Wood Plotting tool based on Python that tries to be efficient and fun while being easy to pick up. As the software is still under development, feedback is highly appreciated.
