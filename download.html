Download

Here you can download the software as well as supplementary data. The different software versions and the icon, which is not necessary but allows to distinguish the program from other python programs in the taskbar, are found under Software. Installation Scripts for the dependencies, which are only useful for the .pyw version, are found under Install Scripts. Additionally sample data is provided for testing under Sample Data.
