Download

Here you can download the software as well as supplementary data. The different software versions and the icon (which is not required, but provides a nice taskbar icon) are found under Software. Installation Scripts for the dependencies (only useful for the .pyw version) are found under Install Scripts. Additionally, sample data is provided under Sample Data for testing purposes.
Specific versions of the software can be found under older releases.

Certain browsers will issue a warning when downloading the .exe file. Should your browser prevent you from downloading the .exe file, you can instead download the zipped version of the latest release under older releases.
